飞网本次简单分享几个自己经常在Google chrome上听的网络电台MMS地址。

先普及下MMS的概念，此MMS不是指“Multimedia Message Service”，这是我们手机上的“多媒体短信服务”，直白的说就是短信。此处的MMS 为(Microsoft Media Server Protocol)，中文“微软媒体服务器协议”，用来访问并流式接收 Windows Media 服务器中 .asf 文件的一种协议，该协议用于访问 Windows Media 发布点上的单播内容。如杭州交通经济广播91.8的MMS地址为mms://vod.hcrt.cn/fm918，即表示FM91.8的Windows Media 服务器名为vod.hcrt.cn，发布点名为fm918。

杭州交通经济广播FM91.8
mms://vod.hcrt.cn/fm918

我喜欢的栏目中午于虎主持的《我的汽车有话说》，不过几次感觉于虎有点经验过于丰富的感觉，有时候打进电话的听众还没有把问题说清楚，他就开始高谈阔论了，稍微有点过。另外诟病的是该台广告超多。
